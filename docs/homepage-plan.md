## 十五、页面结构建议

### 15.1 推荐路由结构

```
/                           → 首页 (文章列表)
/page/:page                 → 首页分页
/category/:slug             → 分类页
/tag/:slug                  → 标签页
/article/:id                → 文章详情
/about                      → 关于页
/archive                    → 归档页
/timeline                   → 时间线
/portfolio                  → 作品集 (可选)
/search                     → 搜索结果页
```

```sql
ALTER TABLE articles ADD COLUMN published_at TIMESTAMP;

ALTER TABLE articles ADD COLUMN seo_description TEXT;
```

### Like/Bookmark 表

```sql
-- 点赞表
CREATE TABLE likes (
  id UUID PRIMARY KEY,
  article_id UUID REFERENCES articles(id),
  user_id UUID REFERENCES users(id),
  created_at TIMESTAMP DEFAULT NOW(),
  UNIQUE(user_id, article_id)
);

-- 收藏表
CREATE TABLE bookmarks (
  id UUID PRIMARY KEY,
  article_id UUID REFERENCES articles(id),
  user_id UUID REFERENCES users(id),
  created_at TIMESTAMP DEFAULT NOW(),
  UNIQUE(user_id, article_id)
);

-- 文章浏览记录 (用于统计)
CREATE TABLE article_views (
  id UUID PRIMARY KEY,
  article_id UUID REFERENCES articles(id),
  user_id UUID REFERENCES users(id),
  ip_address VARCHAR(50),
  user_agent TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);
```
