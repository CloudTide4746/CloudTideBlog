## 一、仪表盘 (Dashboard) 功能扩展

### 1.1 核心统计卡片

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **总文章数** | 显示所有已发布文章数量 | P0 |
| **总浏览量** | 所有文章的累计阅读次数 | P0 |
| **总评论数** | 所有文章收到的评论总数 | P0 |
| **总点赞数** | 所有文章收到的点赞总数 | P1 |
| **今日访问量** | 当日独立访客数 (UV) | P1 |
| **本月新增文章** | 本月发布的文章数量 | P1 |
| **平均阅读时间** | 文章平均阅读时长 | P2 |
| **完读率** | 用户读完整篇文章的比例 | P2 |

### 1.2 数据可视化

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **浏览量趋势图** | 最近7天/30天的浏览量折线图 | P1 |
| **文章发布趋势** | 每月发布文章数量的柱状图 | P1 |
| **分类分布饼图** | 各分类文章数量占比 | P1 |

| **评论增长趋势** | 每日评论数量的折线图 | P2 |
| **流量来源分析** | 直接访问/搜索/引荐占比 | P2 |

### 1.3 快捷操作面板

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **快速创建文章** | 一键跳转到文章编辑器 | P0 |

## 附录：技术实现建议

### 路由结构建议

```typescript
// /admin 路由结构
/admin              → Dashboard (仪表盘)
/admin/articles     → ArticleList (文章列表)
/admin/articles/new → ArticleEditor (新建文章)
/admin/articles/:id → ArticleEditor (编辑文章)
/admin/comments     → CommentList (评论管理)
/admin/users        → UserList (用户管理)
/admin/media        → MediaLibrary (媒体库)
/admin/analytics    → Analytics (流量分析)
/admin/settings     → Settings (博客设置)
/admin/settings/security → SecuritySettings (安全设置)
/admin/settings/appearance → AppearanceSettings (外观设置)
/admin/logs         → OperationLogs (操作日志)
/admin/backup       → Backup (备份导出)
```

### 数据库表扩展建议

```sql
-- 评论表扩展
ALTER TABLE comments ADD COLUMN status VARCHAR(20) DEFAULT 'pending'; -- pending/approved/spam
ALTER TABLE comments ADD COLUMN parent_id UUID REFERENCES comments(id);

-- 用户表扩展
ALTER TABLE users ADD COLUMN role VARCHAR(20) DEFAULT 'user'; -- user/moderator/admin
ALTER TABLE users ADD COLUMN avatar_url TEXT;
ALTER TABLE users ADD COLUMN last_login TIMESTAMP;

-- 操作日志表
CREATE TABLE operation_logs (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  action VARCHAR(50),
  target_type VARCHAR(50),
  target_id UUID,
  details JSONB,
  created_at TIMESTAMP DEFAULT NOW()
);

-- 收藏表
CREATE TABLE bookmarks (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  article_id UUID REFERENCES articles(id),
  created_at TIMESTAMP DEFAULT NOW(),
  UNIQUE(user_id, article_id)
);
```
